## Build Setup

``` bash
# 安装 脚手架vue-cli
npm install --global vue-cli

# 创建一个基于 webpack 模板的新项目
vue init webpack my-project

cd my-project

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
